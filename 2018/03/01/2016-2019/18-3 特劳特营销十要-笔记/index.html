特劳特营销十要-笔记

Created2018-03-01|Updated2025-04-22|2016-2019
|Post Views:

一、营销的本质

1. 营销的本质是什么？

营销就是对思想、产品及劳务进行设计、定价、促销及分销的计划和实施的过程，从而产生满足个人和组织目标的交换。换句话说，营销就是要确定为了销售产品和提供服务以获得利润都需要做哪些事情。

营销工作的职责是发现新理念并将其用语言描述出来，同时将其融入完善的营销战略中，然后释放出它的能量。

2. 预算的基本原则

第一步，根据每种产品的营销生命周期制定市场营销计划。

第二步，评估产品机会。

第三步，安排广告工作——花足够的钱把你的故事讲给目标市场听。

第四步，资金不足时应该立刻停止。将各个项目按照获利可能性和工作有效性排出优先次序，从高往低开始分配㔍。当资金达到极限时，就必须将多余的项目舍弃。

实施好的概念胜过拥有好的概念。——罗伯特 霍尔

保持联系：赢得一个新客户永远比维护已有客户的代价高很多。

3. 分销的重要性：

1）越直接越容易控制

2）不要与客户竞争——好的分销渠道要求你对渠道保持忠诚

3）使产品对分销商的客户更具吸引力

二、如何打造品牌——建立品牌，即在顾客的心智中实现产品和公司的差异化

1. 产品制造于工厂，而品牌是在心智中创建的。

2. 最好的品牌名称是与产品的优势卖点相关联的。

3. 打造品牌即为产品或公司在其所属的领域建立差异化。

4. 品牌是一种承诺，它建立了产品所必须实现的预期。

5. 专注：首先是产品舍弃，也就是说要将精力集中在一种产品上；其次是特性舍弃，也就是说要将精力集中在一种产品特性上；最后是目标市场舍弃，也就是说要将精力集中在某一品类的一个目标细分市场上，这样才有能力成为该细分市场备受青睐的产品。

6. 打造品牌就是将特定品牌联通其差异化根植于顾客的心智中。诀窍就是把精力集中于品牌的核心特征上，而不是过于贪婪。

三、产品战略——成功的产品应该在品类中数一数二，否则就必须开创一个新的子品类

你所需要的是第一或第二的品牌，或者是一个全新的子品类品牌。如果保持强有力的替代战略，也可以称为非常成功的第二产品。

四、如何正确定价——价格是顾客愿意为你的差异化支付、竞争对手允许你索要的价值

1. 必须保证产品价格合理

2. 人们愿意为可感知的价值多付些钱

3. 高质量产品理应价格更高

4. 高价位产品应该彰显声望

5. 高价格和高利润吸引竞争对手

6. 几乎没有公司能靠低价取胜

7. 随着生产量的增加、货币贬值以及竞争的加剧，旧有的规则发生了变化，价格会呈下降趋势，尤其是在商品品类上。此时需要新的战略，比如以独特的方式增加价值和降低成本。

8. 制定价格的关键是弄清楚顾客愿意为你的差异化或附加价值支付多少。

五、增长

成功的公司从来不为增长而苦恼，它们只为如何在市场中取得成功而苦恼。专注主业，专注认知，面对现实。

六、市场调研

  1. 好的市场调研是简单而非复杂的，是关于认知的，而且具有很强的常识性

  2. “差异化实验”——列出某一品类的基本属性，然后让人们按1-10分的等级打分。这种做法是以竞争对手为基础，目的是了解在某一品类中，各方所拥有的概念和认知。

  3. 弄清楚品牌想在潜在顾客的心智中拥有怎样的特性或差异化。市场调研应该提供消费者心智地图，帮助了解竞争对手的认知。

七、如何评估广告

1. 任何广告计划都应从你想传达的产品差异化开始——为什么要购买我的产品而不是其他人的产品。

2. 广告的要点在于——坦诚且有新闻价值。

3. 广告被人们记住是需要时间的，因此必须将你的信息保留较长一段时间，以待人们接受它。

4. 寻找显而易见的概念，它往往是简单、普通、明确，缺乏想象力的。

5. 好的广告将产品的差异点戏剧化，为潜在顾客提供购买理由。

八、选择合适的媒体

1. 在广告业，一个简单的衡量标准是一种媒体的受众数量

2. 要尽可能地使用有声媒体。大脑靠耳朵运转，有声媒体比无声媒体更有力。

3. 公关能帮你推出新理念，赋予其可信度，并制造话题；广告能够迅速为这一理念建立认知度；展览会能够在行业内或产品分销渠道中引起兴奋；直邮可以将你的理念推荐给你最好和需求量最大的那一部分顾客；促销能够吸引人们试用新产品。

4. 首先，应确保我们拥有差异化信息；其次，通过公关将其推出；再次，将我们的资金投入到有声媒体上；最后，不要把促销做过头。

九、公司标志

1. 品牌的力量在于名称而不是视觉符号。名字比符号有力得多，商标应该围绕名字来设计。

2. 必须容易理解

3. 色彩能称为区隔品牌的一种强有力的方式。不要选择竞争对手的色彩。

4. 昵称是市场所赋予你的，是人们主动用来称呼你的，不要试图勉强使用一个昵称。

十、最常见的七个错误

1. 自大：聪明的营销人员能够像潜在顾客那样思考，他们把自己摆在顾客的位置上，不会将自己的主观认识强加到客观环境上。

2. 贪婪

3. 无知：重要的不是更好的产品，而在于更好的认知。

4. 妄想：重要的不是“你想做什么”，而是“你能做什么”。

5. 失去焦点：专注核心业务

6. 修修补补：不要盲目改进

7. 骄傲：不要低估竞争对手
